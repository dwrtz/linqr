Linqr is a Chrome browser extension. It adds a button next to the address bar that when clicked, loads a QR code image from qrcode.kaywa.com that links to the URL of the page you're currently viewing. 

Linqr is an unpacked extension. Here's how you install it...
1.	Create a folder somewhere on your computer and download the files to this folder.
2.	To load the extension , start Chrome and bring up the extensions management page by clicking the wrench icon and choosing Tools > Extensions.
3.	If Developer mode has a + by it, click the + to add developer information to the page. The + changes to a -, and more buttons and information appear.
4.	Click the Load unpacked extension button. A file dialog appears.
5.	In the file dialog, navigate to the folder you created in step 1 and click OK.
